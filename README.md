# React Props Drilling Project (Job Cards UI)

This is a mini React project demonstrating props drilling by passing data through multiple components.
The project renders job cards dynamically using an array of objects.

---

## Features

* Data managed using an array of objects
* Props drilling across multiple components
* Reusable Card component
* Simple and clean UI
* Built with Vite

---

## Tech Stack

* React
* Vite
* CSS

---

## Concepts Used

* Props in React
* Props Drilling
* Component Reusability
* Dynamic rendering using `.map()`

---

## Project Setup

Clone the repository:

```bash
git clone https://github.com/suryansh-mehra/react-props-drilling-demo
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
